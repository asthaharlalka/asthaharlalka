Seasonality Optimization

SELECT dayofweek, AVG(conversion_rate)
FROM campaign_data
WHERE campaign_date >= '2025-01-01'
GROUP BY dayofweek;
Key Findings
• Wed-Fri: 12% higher performance
• Monday: Lowest conversion rates
• Weekend: Moderate performance
Recommendation: Increase mid-week bids by 15%, reduce Monday spend by 20%

Audience Overlap Optimization

WITH platform_users AS (
  SELECT customer_id, platform
  FROM touchpoints
)
SELECT COUNT(*) as overlap
FROM platform_users p1
JOIN platform_users p2 ON p1.customer_id = p2.customer_id;
Overlap Analysis
• Meta-Google: 34% overlap
• Reddit-TikTok: 67% overlap
• Cross-platform frequency: 2.3x
Recommendation: Implement frequency capping across Reddit-TikTok to reduce waste

Conversion Path Optimization

SELECT path, COUNT(*) as frequency,
  AVG(conversion_value) as avg_value
FROM customer_journeys
WHERE conversion_value > 0
GROUP BY path
ORDER BY frequency DESC;
Top Conversion Paths
• Meta → Google: 34% of conversions
• Google only: 28% of conversions
• Meta → Reddit → Google: 12%
Recommendation: Optimize for Meta→Google path with sequential messaging
